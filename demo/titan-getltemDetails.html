getltemDetails

This method retrieves detailed information about files and folders. When cId is not empty, it fetches file information; when groupId is not empty, it fetches folder information.

Function Signature

const data = await getltemDetails(options);

Example


const fileInfo = await getltemDetails({ cId: 12345 });
if (fileInfo.code === 0) {
    console.log('File information:', fileInfo);
} else {
    console.error('Failed to get file information:', fileInfo);
}

const folderInfo = await getAssetGroupInfo({ groupId: 67890 });
if (folderInfo.code === 0) {
    console.log('Folder information:', folderInfo);
} else {
    console.error('Failed :', folderInfo);
}

Parameters

  • cId: Number - The CID of the file. This parameter is optional; if it is empty, the method will fetch folder information based on groupId.
  • groupId: Number - The ID of the folder. This parameter is optional; if it is empty, the method will fetch file information based on cId.

Returns

This function returns a Promise that resolves to an object containing the following information:

  • code: A number representing the result of the operation (0 for success, other values indicate failure).
  • data: Contains the detailed information of the requested file or folder.
  • msg: A string describing the result of the operation, especially in case of errors.

Error Handling

If retrieving information fails, code will be a non-zero value, and message will provide details about the error, such as the file or folder not existing or network issues.

Important Note

Ensure that at least one of cId or groupId has a valid value before calling this method to avoid unnecessary errors.
