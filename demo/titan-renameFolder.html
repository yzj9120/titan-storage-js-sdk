renameFolder

This method allows you to rename a group by specifying its group ID and the new name. It validates the provided inputs before sending the request to the server.

Function Signature

const result = await renameFolder({ groupId, name });

This method sends a request to rename an existing group on the server.

Example

const result = await renameFolder({ groupId: 10, name: 'Updated Group' });
if (result.code === 0) {
    console.log(result); // Success
} else {
    console.error(result); // Failure
}

Parameters

  • groupId: Number - The ID of the group to rename. This is a required parameter and must be a valid group ID.
  • name: String - The new name for the group. This is a required parameter and must be a valid string.

Returns

This function returns a Promise which resolves to an object containing:

  • code: A number representing the result of the operation (0 for success, other values indicate failure).
  • data: The updated group object if the operation is successful.
  • msg: A string describing the outcome of the operation, especially in case of errors.

Error Handling

If the rename operation fails, the code will be non-200, and the message will provide details about the error, such as an invalid group name, invalid group ID, or server-related issues.

Important Note

Ensure that both groupId and name parameters are valid before calling the method to avoid unnecessary errors. The method uses validators to check the input values, and it will return any validation errors if the inputs are invalid.
